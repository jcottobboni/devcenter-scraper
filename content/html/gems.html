Ruby gems can be installed on Heroku via Bundler and an associated Gemfile.

Almost any gem - even those with native dependencies - can be installed using Bundler. If there’s a specific gem that won’t install on Heroku, please submit a support ticket.

See the gem manifest page for full details on the deprecated .gems support.

If you have questions about Ruby on Heroku, consider discussing it in the Ruby on Heroku forums.
